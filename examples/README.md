# Examples

This folder holds **optional** reference snippets for customizing Agents Studio. For the full doc set, see [../docs/README.md](../docs/README.md).

- **`assistants-extended.sample.yaml`** — sample shape for extra agents and keywords. Copy ideas into `apps/agent-core/assistants.yaml` (or edit via the Studio UI) and adjust models to match your provider.

These files are not loaded automatically; they document patterns only.
